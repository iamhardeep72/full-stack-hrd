# 🚀 Full Stack Todo App (React + Node.js)

## 📌 Overview

This is a simple full-stack Todo application built with:

* Frontend: React
* Backend: Node.js + Express

---

## 📁 Project Structure

```
client/   → React frontend
server/   → Node.js backend
```

---

## ⚙️ Prerequisites

* Node.js (v18 or higher)
* npm or yarn

---

## ▶️ Run Backend (Server)

```bash
cd server
npm install
npm start
```

Server will run on:
👉 http://localhost:5000

---

## ▶️ Run Frontend (Client)

Open new terminal:

```bash
cd client
npm install
npm run start
```

Frontend will run on:
👉 http://localhost:5173 (Vite default)

---

## 🔗 API Endpoints

* GET `/api/todos` → Get all todos
* POST `/api/todos` → Add new todo

---

## 🧪 Run Tests (Backend)

```bash
cd server
npm test
```

---

## 🛠 Notes

* Make sure backend is running before frontend
* API URL is set to:

```
http://localhost:5000
```
